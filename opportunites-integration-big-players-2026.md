## 1. Google — Universal Commerce Protocol (UCP) ★★★★★

### C'est quoi
Un **protocole ouvert pour le commerce agentique** co-développé par Google avec Shopify, Etsy, Wayfair, Target, Walmart. Il permet à des agents IA (Gemini, Google AI Mode, etc.) de découvrir des produits, gérer un panier, et compléter un checkout directement sur les surfaces Google.

### Pourquoi c'est gros
- Compatible avec les standards existants : **A2A** (Agent2Agent), **AP2** (Agent Payments Protocol), **MCP** (Model Context Protocol)
- Endorsé par 20+ géants : Adyen, American Express, Best Buy, Flipkart, Macy's, Mastercard, Stripe, The Home Depot, Visa, Zalando
- **Checkout natif** dans Google AI Mode / Gemini via Google Pay et bientôt PayPal
- Nouvelles capacités (mars 2026) : **Cart** (panier multi-produits), **Catalog** (infos produits temps réel), **Identity Linking** (avantages fidélité cross-plateforme)

### Comment tu fais de l'argent
- **Service d'onboarding UCP** : Configurer le feed produit du client dans Google Merchant Center avec les nouveaux attributs conversationnels
- **Structuration catalogue enrichi** : Ajout des attributs UCP (questions fréquentes, accessoires compatibles, substituts) — va au-delà du feed produits basique
- **Intégration Identity Linking** : Connecter le programme fidélité du client à UCP pour que les avantages membres s'appliquent sur les surfaces Google
- Partners d'implémentation annoncés : Commerce Inc, Salesforce, Stripe

### Lien
- Spec : https://ucp.dev/
- Onboarding Merchant Center : https://support.google.com/merchants/answer/16837055

- **Setup Buy for Me** : S'assurer que le site e-commerce du client est compatible avec l'agent IA d'Amazon qui complète l'achat
- **Monitoring et analytics** : Suivi des conversions provenant de Shop Direct

## 6. WordPress.com — Agents IA via MCP ★★★☆☆

### C'est quoi
WordPress.com permet maintenant aux **agents IA** (Claude, ChatGPT, Cursor, etc.) de **créer, éditer et publier du contenu** sur les sites des clients via le Model Context Protocol (MCP). Les agents peuvent aussi gérer les commentaires, le SEO (alt text, méta descriptions), et les catégories.

### Pourquoi c'est intéressant
- WordPress alimente **43% de tous les sites web**
- Les changements nécessitent l'approbation du propriétaire (drafts par défaut)
- L'agent analyse le thème et design du site avant de créer du contenu (couleurs, polices, espacement)
- Config via wordpress.com/mcp — toggle les capacités souhaitées

### Comment tu fais de l'argent
- **Setup MCP + Agent IA** : Connecter le site WordPress du client à un assistant IA, configurer les permissions
- **Stratégie de contenu automatisé** : Définir les workflows de publication IA (drafts → review → publish)
- **SEO IA** : Utiliser l'agent pour nettoyer alt text, méta descriptions, catégories sur tout le site
- Bon **add-on** à ton service principal ACP : "On intègre votre catalogue dans ChatGPT ET on automatise la création de contenu marketing via IA"

### Comparaison technique détaillée

#### 1. ACP vs Google UCP — Les deux poids lourds ouverts

| Aspect | ACP (OpenAI) | UCP (Google) |
|---|---|---|
| **Philosophie** | Protocole ouvert, infrastructure entre marchands et ChatGPT | Protocole ouvert co-développé avec l'industrie pour tout l'écosystème agentique |
| **Spec feed** | Format propriétaire ACP : JSON lines, 56 champs dont 27 requis (id, title, description, url, variants, price, availability...) | Basé sur Merchant Center existant + nouveaux attributs conversationnels (FAQ, accessoires, substituts) |
| **Méthodes d'ingestion** | File Upload (snapshot) + API temps réel (upsert produits, promotions) | Feed Merchant Center + API Merchant Center |
| **Capabilities** | Découverte produits, variantes, promotions | Découverte + Cart (panier multi-items) + Catalog (infos temps réel) + Identity Linking (fidélité) |
| **Checkout** | Redirige vers le site marchand (abandon de l'Instant Checkout dans ChatGPT) | Checkout natif dans AI Mode/Gemini via Google Pay + PayPal |
| **Écosystème de compatibility** | ACP seul (mais open-source) | Compatible A2A + AP2 + MCP — interopérable par design |
| **Partners checkout** | Stripe uniquement | Google Pay, PayPal, bientôt plus |
| **Impact pour ton business** | Ton offre principale. Feed format spécifique. Application sur waitlist. | **Le complément #1.** Même données de base, adaptation des attributs. Travail incrémental faible si ACP déjà fait. |

**Verdict :** UCP est plus mature au niveau écosystème (20+ partners dès le jour 1, compatible multi-protocoles). ACP a l'avantage de l'audience ChatGPT ultra-qualifiée et de l'intention d'achat directe. **Les deux sont complémentaires, pas concurrents** — même feed de base, deux canaux IA.

**Effort incrémental pour ajouter UCP si ACP déjà fait :** ~30-40% du travail ACP initial (adaptation des attributs + onboarding Merchant Center).

#### 2. ACP vs Amazon Shop Direct + Buy for Me

| Aspect | ACP (OpenAI) | Amazon Shop Direct |
|---|---|---|
| **Contrôle marchand** | Total — le marchand contrôle prix, présentation, fulfillment | Partiel — Amazon affiche les produits dans ses résultats, mais le marchand garde le checkout |
| **Checkout** | Sur le site du marchand | L'IA Amazon complète l'achat SUR LE SITE DU MARCHAND au nom du client (Buy for Me) |
| **Feed** | Format ACP (JSON lines) | Via partenaires feed : Feedonomics, Salsify, CedCommerce |
| **Données partagées** | Catalogue → OpenAI pour indexation | Catalogue → Amazon pour indexation + insights concurrentiels |
| **Risque** | Faible — OpenAI ne vend pas de produits | **Moyen-Élevé** — Amazon utilise les données pour identifier des tendances et potentiellement concurrencer |
| **Audience** | Utilisateurs ChatGPT (haute intention IA) | Acheteurs Amazon (haute intention achat) |

**Verdict :** Amazon apporte un trafic d'acheteurs massif mais **le partage de données avec Amazon est un risque stratégique** (ils peuvent s'inspirer de ton catalogue pour lancer leurs propres produits). ACP est plus sûr pour le marchand. Bon complément si le client vend déjà sur/avec Amazon. 

**Argument de vente :** *"ACP vous protège — OpenAI ne vend pas de produits. Amazon, oui."*
